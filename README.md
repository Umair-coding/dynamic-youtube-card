# 🎥 Dynamic YouTube Card Generator

A simple JavaScript project that dynamically creates a YouTube-style video card using DOM Manipulation.

---

## 🚀 Features

- 🎬 Generates a YouTube-style video card dynamically
- 🖼 Displays video thumbnail
- 📝 Displays video title
- 📺 Displays channel name
- 👀 Displays views and upload time
- ⏱ Displays video duration
- ⚡ Uses JavaScript functions and DOM Manipulation

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

---

## 📚 Concepts Practiced

- DOM Manipulation
- Functions
- Parameters
- Template Literals
- innerHTML
- querySelector()
- Dynamic HTML Generation
